# AI-Powered Personalized News Feed

## Project Overview
This project aims to build a web-based news feed system that curates and displays personalized content based on user preferences and browsing behavior.

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Python (FastAPI/Django/Flask)
- Database: PostgreSQL, MongoDB
- AI/ML: TensorFlow, Scikit-learn

## API Used
- NewsAPI: https://newsapi.org/
